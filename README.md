## Requirements
- Computer with python 3
- WeMos board (or any other esp8266 with WiFi)
- Motor shield
- Servo motor
- Motor
- Battery pack 
- Jumper cables (M-F & M-M)
- Lego Technic

## Installation
Install the two libraries: socket, pygame, esptool

```bash
pip install socket
pip install pygame
pip install esptool
```
